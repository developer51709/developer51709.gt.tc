Random Facts / Jokes / Quotes API Documentation

Overview

This API returns a random fact, joke, or quote. Choose the type via a parameter and receive a single item in JSON.

Endpoint

http://developer51709.gt.tc/api/random.php

Parameters

  • type: fact | joke | quote (default: fact)

Examples

Get a random fact

http://developer51709.gt.tc/api/random.php?type=fact

Get a random joke

http://developer51709.gt.tc/api/random.php?type=joke

Get a random quote

http://developer51709.gt.tc/api/random.php?type=quote
